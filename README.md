# Task Manager
A desktop window with various time tracking apps and a tree which grows based on tasks completed.

<img width="1464" height="740" alt="Screenshot 2026-07-11 at 10 23 13 PM" src="https://github.com/user-attachments/assets/5df941ee-d1c8-400d-ae8d-4d45e520bac8" />

Try It Live: https://rushil070.github.io/Thingondesk/

Code: https://github.com/rushil070/Thingondesk

- Displays a live EST clock and today's date in the top bar.
- Includes desktop apps for Calendar, Notes, Timer, and Tasks, each opening in its own draggable window.
- Lets users create and complete tasks, with a tree that grows through five stages as more tasks are completed.
- Includes a toggle background button which swaps between light and dark mode backgrounds.
